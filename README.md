# Avani — Portfolio (React)

Built with React + Vite, plain JavaScript (no TypeScript).

## Running it locally

```
npm install
npm run dev
```

Then open the local URL Vite prints in your terminal.

## Building for production

```
npm run build
```

The output goes to the `dist` folder — upload that anywhere (Netlify, Vercel, GitHub Pages, etc).

## Folder structure

- `src/components` — one component per section of the page
- `src/styles` — one CSS file per component, plus `globals.css` and `variables.css` for shared stuff
- `src/assets` — drop images/icons here if you add any later
